# Agents RU

## Hero

### Label

Агенты

### Title

AI-агенты под задачи, где бизнесу нужен результат

### Body

Каждый агент проектируется под конкретный процесс: что нужно сделать, какие
данные нужны, какие правила соблюдать, где подключается человек и какой
результат должен получить бизнес. Поэтому AI становится рабочим инструментом
внутри процесса, а не отдельным интерфейсом без привязки к задаче.

## Agent Cards

### Агент бизнес-процесса

Берет на себя повторяющиеся этапы: собрать данные, проверить условия,
подготовить следующий шаг и передать результат дальше.

### Агент клиентского сервиса

Работает по материалам компании, закрывает типовые вопросы и передает сложные
обращения сотрудникам без потери контекста.

### Агент для внутренних команд

Помогает сотрудникам быстро находить ответы в регламентах, инструкциях,
документах и внутренних правилах.

### Агент для документов

Разбирает документы, извлекает важные данные, сверяет их с правилами и готовит
результат для проверки человеком.

### Агент контроля и compliance

Поддерживает процессы, где важны правила, проверки и аккуратная работа с
чувствительными данными, например AML-сценарии в банках.

### Отраслевой AI-агент

Настраивается под специфику бизнеса: финансы, сервисы, операционные команды,
экспертизу, документы или внутренние процедуры.

## Spotlight

### Title

Первый сценарий выбирается по измеримому эффекту

### Text

Первым может быть AI-консультант на сайте, внутренний помощник по регламентам,
агент для документов или AML-сценарий для банковской команды. Мы выбираем
процесс, где быстро видно снижение ручной нагрузки, ускорение работы или
повышение качества контроля.

## Checklist

- разбирает входные данные
- работает с материалами компании
- проверяет условия и правила
- готовит результат для сотрудника
- передает сложные случаи человеку
- снижает ручную нагрузку
- создает основу для следующих сценариев
