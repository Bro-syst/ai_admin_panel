# Client Panel Overview

`apps/client-panel` is the user-facing frontend application workspace.

Current scope:
- Vite/React/TypeScript baseline;
- root route `/`;
- not found route;
- test/lint/build scripts;
- local dev server on port `5174`.

The app does not yet contain product flows. Add routes and modules only after the client scenario, backend contract, and UI states are clear.
