# Packages

Shared frontend packages live here when reuse is proven across more than one app.

Do not extract code here only because it might be useful later. Start inside an app, then move to `packages/` when at least two apps need the same contract.

Likely future packages:
- `shared-ui`
- `shared-config`
- `shared-api`
- `shared-auth`
- `shared-i18n`
